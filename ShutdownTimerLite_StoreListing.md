## Description (max 10,000 characters)

```
Shutdown Timer Lite is a compact, free desktop utility that lets you schedule automatic power actions on your Windows PC. Set a countdown timer and walk away — your computer will shut down, restart, sleep, hibernate, or log off when the timer reaches zero.

FEATURES

• Countdown Timer — Set hours, minutes, and seconds for precise scheduling
• 5 Power Actions — Shutdown, Restart, Sleep, Hibernate, and Log Off
• Safety Grace Period — A 5-second countdown before destructive actions (shutdown, restart, hibernate, log off) gives you time to cancel
• System Tray — Minimises to the notification area so it stays out of your way while counting down
• Live Countdown — The Start button transforms into a live HH:MM:SS display while running
• Single Instance — Prevents duplicate timers from running simultaneously
• Dark Theme — Modern dark interface that's easy on the eyes

SIMPLE BY DESIGN

No account required. No internet needed. No data collected. Just set your timer and go.

Shutdown Timer Lite stores absolutely no personal data. All it does is count down and perform the power action you chose. See the Privacy Policy for full details.

WANT MORE?

Upgrade to Shutdown Timer (Pro) for advanced features:
• Process Monitoring — Trigger actions when a specific app closes
• Idle Detection — Act when your PC has been idle for a set time
• Daily/Weekly Scheduling — Recurring power schedules with day-of-week selection
• Quick Actions Dashboard — One-click sleep, restart, or shutdown from a live overview
• Pre-action Warning Banner — An on-screen warning with cancel button before any action fires
• Customisable Warning Time — Set your own warning countdown (5–300 seconds)
• Theme Support — System, Light, or Dark theme

SYSTEM REQUIREMENTS

• Windows 10 version 1809 (build 17763) or later
• Windows 11 supported
• x64 or ARM64 processor
• ~25 MB disk space
```
